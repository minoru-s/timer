# Focus Timer

**URL**: https://minoru-s.github.io/timer/

「急かされない」をモットーにしたミニマルなタイマーアプリです。就職活動における適性検査（SPIや玉手箱など）の問題集を解く用途を想定して作成しました。

## 特徴

- **秒数非表示**: 計測中は具体的な秒数が出ないため、数字に急かされず問題に集中できます。
- **ワンタップ開始**: 30秒、45秒など、適性検査でよくある制限時間がプリセットされており、ボタンを押すだけで開始できます。
- **結果表示**: 完了ボタンを押すと、規定タイムに対してどれくらいのペースで解けたかが表示されます。
- **PWA対応**: スマホやPCのホーム画面に追加してアプリのように使用できます（オフライン対応）。
